Live preview

Piskel gives you a live preview of your work.
You no longer have to guess how it will look like when it's done : you see it !

Export to GIF, PNG...

Several export modes supported. Animated GIF to share the animations, spritesheet PNG/ZIP for game projects...

Your gallery

Keep track of all the sprites you created.
Everyone can see your public gallery, but you can keep any sprite you want private.

Open Source

All the code is open-source and available on GitHub.
http://github.com/juliandescottes/piskel

Offline versions

Free desktop & offline applications are available for Windows, OSX and Linux. Checkout the download page.
